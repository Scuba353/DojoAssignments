.hide and .show

When the hide button is clicked this text will be hidden. Then when the show button is clicked the text will reappear!

.toggle

When toggle button is clicked, the text will alternate between appear and hidden.

.slideDown and .slideUp

When the slideDown button is clicked this text will slide down and appear from an apparent top. Then when the slideUp button is clicked the text will slide towards the top and disapear!

.slidetoggle

When toggle button is clicked the text will alternate between slide up and slide down.

.fadein and .fadeout

When fade in is clicked the text will fade in and when fade out the text will fade out and disapear.

.addclass

Add a class attribute to the called element in this case the headers and then you can add css styling to the new class. Helpful in adding the same class to an element throughout a document.

.before

When the button is clicked new text is added before the element tag as a new element.

.after

When the button is clicked new text is added after the element tag as a new element.

.append

When the button is clicked new text is added to the current element called on. It is addeded directly to the end of the current element.

.html

When this button is clicked the content of the HTML element is changed.

.attr

Hover over the picture for the alternate picture atribute to appear.

.val

When text is typed in the text box, an alert will be triggered when the button is clicked indicating the data typed in the text box. The val set inputs data into the text box.

.text

When set button is clicked the text of the selected element will be set to new text. When the get button is clicked it will grab the text of another element such as all of the headers that are returned in the alert!
